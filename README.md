# StockFlow
Ferramenta de gestão de veículos feito com o foco no aprendizado e estudo.

# Funcionalidades
* Controle e edição de veículos
* Cadastro de veículos
* Cadastro de funcionários
* Dashboard do estoque
* Geração de contrato
* Edição de funcionários
* Login de funcionários
* Adição e edição de gastos do veículo
* Visualização de veículos vendidos
* Visualização de veículos disponíveis e indisponíveis
* Troca de senha
